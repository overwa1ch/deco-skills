# Shot Table Prompt Template（分镜表固定提示词）

仅在用户选择分镜表时使用。输出【分镜表生成模版】和单独的【短任务提示词】，逐字照发；只把 `SEGXX` 替换为当前 SEG 序号，不得增删、改写、扩展、解释或附加任何内容。不加样式提示词——分镜表调用绘图模型自身的"分镜表"概念。

上传材料后，先发送【分镜表生成模版】，再单独发送【短任务提示词】。

```text
【分镜表生成模版】

你绘制分镜表，我来审查。

请读取我上传的材料：
1. 故事 / 剧本 / 导演脚本等材料：作为当前分镜表生成依据。
2. 静态资产（如有）：作为人物、空间、比例和道具参考。

不要拆分、改写、补全、总结、修复或判断我上传的故事 / 剧本 / 导演脚本等材料；不要新增剧情。
```

【短任务提示词】

```text
用分镜表，把SEGXX（SEG序号）的内容完整的表现出来。用数字标出镜头序号，不需要文字描述。多给我几个版本(分镜表与分镜表之间，不要拼成一张图)。
```

定位：分镜表以模型原生概念完整呈现一个 SEG 的镜头序列——数字标序、无文字说明、无样式约束；与故事板（草图风格 + 景别/运镜/动作标注）各有所长。故事板长于单镜执行指令，分镜表长于序列全貌与镜头覆盖；选择权在用户，无粗略/正式分级。

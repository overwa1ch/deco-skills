# AI Video Composition Rules

Use before writing `processed director script` shots.

## Core Formula

AI video composition = first-frame space + motion path + camera psychological distance + continuity anchor.

Composition is time design. It should tell the image/video model how the frame starts, how space changes, where attention lands, and what stays stable across the cut.

## Shot Composition Contract

Write `构图 / 空间层次` as one compact time-composition sentence. Use only the clauses and layers visible in the shot:

```text
首帧：前景...，中景...，背景...，主体位于...；
路径：主体/镜头/焦点从...到...；
落点：最后停在...；
锚点：保持...不变。
```

Use the full four-part shape only when every named layer is visible and needed for people, blocking, reveal, or continuity.
For close-ups, inserts, or very short shots, write one shorter sentence around the focal subject and visible action. Do not force foreground, midground, and background into every composition.

## What Each Part Does

- `首帧`: establish foreground / midground / background and subject scale or position.
- `路径`: describe subject movement, camera movement, focus transfer, or spatial reveal.
- `落点`: state the final image or attention point at the cut.
- `锚点`: name the stable layout, prop position, light direction, or character position that prevents continuity drift.

## Field Discipline

- Use only source-supported anchors or approved layout anchors.
- Do not write prop inventories.
- Let the crop limit the allowed nouns. A close-up or insert contains the focal subject, visible interacting part, and contacted edge only when needed.
- State whether `路径` belongs to the camera, the subject, or focus. When the camera moves, include its start, physical path, relation to the subject, and landing.
- When no stronger camera path is motivated, use a slow push from the current shot size to a slightly tighter framing.
- Use `固定镜头` only for an explicit user lock or an intentionally still comparison, surveillance, freeze, or deadpan hold. Describe subject or focus movement separately.
- Keep off-frame body parts, clothing, props, background, reflections, and causal sources out of the composition sentence.
- Describe a viewpoint by what the image shows. For a selfie frame, write `自拍角度，手臂伸向镜头`; name a phone only when it is visible.
- For people shots, include position and rough scale when it affects readability.
- For multi-person shots, make near/far, left/right, and relationship distance legible.
- Camera distance and height must serve the beat: close/low/occluded for pressure, near/shallow for secrecy, wide/slow push for isolation, and held static only for deliberate deadpan or comparison.
- Avoid generic quality adjectives; replace them with visible spatial evidence.

## Continuity Use

Across shots, check that the previous landing can become the next first frame.
Track room layout, character position, key prop position, screen direction, and the last action state in validation. Copy an anchor into a shot description only when the frame shows it.

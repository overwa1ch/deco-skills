# Visual Optimization Rules

Use only after the source-preserving `SEG` breakdown is visible and the user agrees to optimize, or when the user explicitly asks for a more visual/directorial version.

## Preservation

Preserve core plot facts, cause/effect, dialogue meaning, character intent, locations, named characters, key props, and beat endings. Get user approval before adding named characters, locations, key props, future turns, or changed endings.

## Optimization Targets

Make each `SEG` easier to direct and storyboard by turning abstraction into visible behavior:

- emotion into posture, breath, gaze, hand tension, distance, hesitation, or repeated micro-action
- relationship into blocking, entrance/exit, interruption, pursuit, obstruction, giving/taking props, or changed eye line
- time pressure into environmental change, sound cue, prop state, or repeated action
- exposition into readable action, object state, body response, or short dialogue anchor

## Director-Script Optimization Priorities

When optimizing toward a director script, prioritize visible execution that can later be assigned to shots without pre-writing final `拍摄手法` camera paths:

1. Visible beat: turn abstract intent into posture, breath, gaze, hand tension, distance, hesitation, object contact, or repeated micro-action.
2. Blocking and object state: make entrances, exits, interruptions, giving/taking props, concealment, reveal, and distance changes physically legible.
3. Attention movement: state what action, sound, focus shift, or reaction moves the viewer's attention; do not write frame-position or composition-layer prose unless the user explicitly asks for a composition-heavy version.
4. Shot task prep: identify the setup, delay, reveal, handoff, reaction, reversal, or cold close that each later shot must land.
5. Continuity anchor: track the layout, character position, prop position, sound source, or light state that connects one beat to the next. Keep it outside a later shot block until that frame can show it.

Replace style adjectives with visible evidence. Load `ai-video-composition-rules.md` only on explicit composition-heavy requests.

Optimization gathers candidate visible beats at `SEG` level. It does not copy every candidate beat or continuity fact into every shot. When assigning a shot, retain only the active subject, visible contact, and necessary visible result inside that crop.

## 空镜 Use

Use 空镜 as story-bearing shots. It must carry place, time, relationship, transition, prop state, or aftermath.

Three useful types:

- 环境空镜: establishes time, space, weather, light, or crowd state.
- 道具空镜: centers a used, mentioned, or relationship-bearing object.
- 状态承接空镜: lets environment or object behavior carry an offscreen character state.

Useful positions:

- before a scene to locate the audience
- after an important line or action to land aftermath
- between scenes to bridge time, space, or relationship continuity

空镜 writing formula:

```text
拍摄主体 + 可见状态 + 动态/声源 + 光线/天气 + 注意力落点
```

Use 空镜 only when it changes understanding of place, time, relationship, prop state, aftermath, or transition.

## 调度 And Action Dynamics

For one-person action, choose the smallest visible action that changes the frame:

```text
主要可见动作 + 必要可见结果
```

Add one secondary change only when it is visible in the same crop and necessary for contact, blocking, focus, or continuity. Use clothing, hair, props, light, sound, and camera movement only when they are visible in the current frame and change story information. Do not preload close-ups with full-person or full-scene detail.

For two-person scenes, clarify:

- who appears first
- who notices whom
- how distance changes
- who changes the other's action
- what blocks, reveals, interrupts, or redirects the relationship

Use concrete actions such as 遮挡、观察、靠近、追赶、绕身、挡路、递交、夺回、后退、停住、转身、避开视线.

## Density

Each `SEG` should be storyboard-ready with one main action chain and a small number of directorial beats. Compress repeated exposition and unrelated emotional inserts.

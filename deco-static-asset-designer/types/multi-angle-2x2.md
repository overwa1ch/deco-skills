# Multi-angle 2x2 四宫格场景参考

仅在对应 `Sxx location_reference` 场景资产图已确认后使用；把确认图作为参考，每个场景单独生成一次。用户可以直接点名本产物；返回九宫格无法保持场景身份、拓扑、家具、设备、光线或风格时，也可用本产物降低信息密度。本产物是独立固定提示词，不适用生产资产字段与固定块契约。

## 固定提示词（逐字，仅替换 XX）

```text
参考场景资产图，生成这个XX场景不同角度不同景别的2×2四宫格场景图。
```

只把 `XX` 替换为当前场景名称。不得增删、改写或扩展其他文字。只输出替换后的单句提示词，不附标题、标签、代码块、元数据、参考图绑定说明、解释或审查内容。

## 评审清单

- 确认：2×2 四宫格成立；四格角度与景别有意义地互不相同；每格保持已确认场景的身份、空间拓扑、建筑、家具、设备、材质、光线与风格。
- 标记：格数错误、覆盖重复、地点重设计、空间矛盾、家具或设备漂移、光线或风格漂移、无关剧情摆拍。

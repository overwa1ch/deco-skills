# Sxx 地点参考（location_reference）

一份 `Sxx location_reference` 锁定一个稳定地点的空间身份，用一个可复用的宽幅建立视角呈现。写产前先读 references/contracts.md。

## 输出模板（按序填写）

1. **地点设计**：生成[地点名称]的地点参考图，[视觉媒介与核心风格]。[地点类型、用途、整体尺度、主要空间关系，以及最能建立地点身份的建筑、陈设、材质和准确标识；只保留影响画面与复用的事实]。
2. **画面要求（固定·几何契约）**：16:9地点参考图，使用单一宽幅建立视角，完整呈现地点尺度和主要空间关系，关键区域无遮挡、不裁切；默认空场景，只有尺度难以判断时加入一个匿名比例人物，无具名角色和剧情动作。
3. **视觉呈现**：[把 regional_anchor、style_aesthetic、时间、天气、光线和空气状态转成可见的地域、时代、色彩、材质与氛围]。
4. **负面提示词**：[按当次真实风险现写，可从下方评审清单选取]。

## 评审清单

- 确认：地点身份连贯；尺度与主要空间关系可读；标志性建筑、陈设与材质史清楚；单一宽幅建立视角；除必要匿名比例人物外空场景。
- 标记：泛化概念图、海报式构图、无依据地标、地点重设计、关键区域被遮挡、具名角色、剧情动作。

## 示例（最终输出形态）

1. **地点设计**：生成县城供销社营业厅的地点参考图，写实电影质感。1985年北方县城供销社营业厅，单层砖木结构约二十米进深，水磨石地面，L形玻璃木柜台沿两面墙展开，柜台后靠墙立到顶的木货架分格摆放布匹、搪瓷缸、暖水瓶和罐头，正门对侧墙上悬挂"发展经济 保障供给"红底白字横幅，柜台一角立老式台秤和算盘，天花板垂下双管日光灯和缠着胶布的电线。
2. **画面要求（固定·几何契约）**：16:9地点参考图，使用单一宽幅建立视角，完整呈现地点尺度和主要空间关系，关键区域无遮挡、不裁切；默认空场景，只有尺度难以判断时加入一个匿名比例人物，无具名角色和剧情动作。
3. **视觉呈现**：冬末上午的冷白天光从临街玻璃窗斜入，与日光灯偏绿的光混合，空气有浮尘；墙面绿漆刷到腰线、以上刷白，漆面起皮，柜台玻璃有磨痕和指印。
4. **负面提示词**：避免现代商品与货架、超市式布局、崭新装修、镜头畸变、无依据地标或多余文字。

平台设置：画幅比例 16:9（在生成平台侧设置；提示词文字不能改变画布比例）

# Pxx-state 道具状态

一份 `Pxx-state` 以一个已批准基础 `Pxx` 为身份与结构基准，只锁定一个剧情关键改变态。生产提示词必须附基础 `Pxx` 参考图；缺图时指出缺失并停止，只有用户明确要求草案时才可标注身份锁未决后起草。写产前先读 references/contracts.md。

## 输出模板（保留字段顺序，不加序号）

**生成对象与风格**：以[道具自然名称的基础 Pxx 参考图]为基准，生成该道具处于[目标状态名称]的道具状态参考资产，[视觉媒介与核心风格]。
**画面布局（固定·几何契约）**：16:9道具状态资产页。左侧为同一道具当前状态的精细3/4特写；右侧为同一道具当前状态的正面、侧面、背面三视图。所有视图完整呈现道具，只呈现同一个目标状态，尺度、结构、材质、颜色、标记和状态变化一致，不生成基准状态或前后对比。
**基础道具参考**：以[对应基础 Pxx 参考图]作为道具身份、轮廓、尺度、结构、机制、材质、颜色、标记、磨损和基准状态的来源。
**目标状态**：[一个明确的损坏、浸湿、耗尽、开启、激活或其他剧情关键状态；写明可见阶段、程度和范围，原因只在用户材料支持时填写]。
**状态变化**：[相对基础参考图写明发生变化的部件、准确位置、方向、范围、程度和当前结果，只写当前视图可见的变化]。
**保持不变**：[只列会影响身份连续性或机制正确性的少量不变量，例如整体轮廓、尺度、未受影响的结构、关键连接、原有材质、颜色或准确标记]。
**材质与物理表现**：[写明目标状态造成的变形、断裂、湿度、残留、耗损、开合、发光或其他可见物理响应，保持材料行为与结构机制可信]。
**状态一致性**：[明确同一目标状态在左侧3/4特写与右侧正、侧、背三视图中的位置、范围、程度和物理结果必须一致，不混入基础状态或另一改变态]。
**标记与文字**：[只在目标状态改变、遮挡或威胁准确文字、符号、图案或标签时填写；其余由基础参考图承载，不生成额外文字或品牌]。
**地域与视觉表现**：[把 regional_anchor 和 style_aesthetic 中与当前道具状态有关的特征转成可见结果，不抹除基础身份、结构、机制或状态差异]。
**光影（固定·呈现契约）**：均匀柔和的摄影棚参考光，道具轮廓、结构、目标状态、材质变化、接缝、标记和使用机制清晰可读，无广告化高光或戏剧性光效。
**背景（固定·呈现契约）**：纯净的中性灰色摄影棚背景，无具体场景、人物、文字说明或装饰。
**负面提示词**：[按当次真实风险现写，可从下方评审清单选取]。

## 评审清单

- 确认：基础 Pxx 身份、轮廓、尺度、结构、机制、材质和准确标记保持连续；全页只有一个目标状态；3/4 特写与三视图中的变化位置、程度和物理表现一致；目标状态清楚可读且不妨碍必要机制判断。
- 标记：缺少基础参考、道具重设计、多种改变态混排、基准状态或前后对比面板、损伤位置漂移、无依据的额外破坏、材料响应失真、机制不可操作、臆造文字或品牌、人物或完整场景摆拍。

## 示例（最终输出形态）

**生成对象与风格**：以白底红花搪瓷暖水瓶基础参考图为基准，生成该暖水瓶处于跌落后右下壶身凹陷并缓慢渗水状态的道具状态参考资产，写实电影质感。
**画面布局（固定·几何契约）**：16:9道具状态资产页。左侧为同一道具当前状态的精细3/4特写；右侧为同一道具当前状态的正面、侧面、背面三视图。所有视图完整呈现道具，只呈现同一个目标状态，尺度、结构、材质、颜色、标记和状态变化一致，不生成基准状态或前后对比。
**基础道具参考**：以白底红花搪瓷暖水瓶基础参考图作为道具身份、轮廓、尺度、结构、机制、材质、颜色、花纹、磨损和基准状态的来源。
**目标状态**：暖水瓶跌落后的稳定结果；壶身右下部一处椭圆形内凹，凹陷中心搪瓷脱落，金属接缝处缓慢渗出清水，其余部位保持原状。
**状态变化**：右下壶身向内凹陷约一个掌心宽，凹陷中心露出暗灰金属，白色搪瓷沿受力边缘形成短裂纹；最低接缝处有一道连续湿痕和少量水滴。
**保持不变**：黑色提梁与把手、木塞口、壶嘴比例、整体高度和未受损一侧轮廓直接延续基础参考图。
**材质与物理表现**：薄钢壶身形成圆钝内凹，脆性搪瓷在受力中心剥落并沿边缘开裂；清水沿重力方向汇到最低接缝，湿痕与水滴透明自然。
**状态一致性**：左侧3/4特写与右侧正、侧、背三视图均保持右下壶身同一处掌心宽凹陷、同一圈短裂纹和最低接缝处的缓慢渗水，不出现完好壶身或第二种损坏状态。
**标记与文字**：壶身原有红花图案的位置、形状和褪色程度直接延续基础参考图；本状态不新增文字、符号或品牌。
**地域与视觉表现**：保留长期家庭使用造成的旧搪瓷磨痕和低饱和写实电影质感，损坏状态清楚但不广告化渲染。
**光影（固定·呈现契约）**：均匀柔和的摄影棚参考光，道具轮廓、结构、目标状态、材质变化、接缝、标记和使用机制清晰可读，无广告化高光或戏剧性光效。
**背景（固定·呈现契约）**：纯净的中性灰色摄影棚背景，无具体场景、人物、文字说明或装饰。
**负面提示词**：避免暖水瓶身份或花纹改变、凹陷位置和程度在各视图漂移、出现多处无依据破坏、同时展示完好版本、渗水方向违背重力、增加人物或家庭场景。

# 产出契约（单一真源）

写任何生产资产提示词前读本文件。类型文件（types/*.md）只含该类型的几何契约、内容槽位、评审清单和金标准示例；所有共享规则以本文件为准，类型文件不得另立或改写。

## 一体化提示词

每份可复制提示词同时包含资产设计规格（身份、结构、比例、材质、状态、视图、防漂移）与图像生成指令（呈现、布局、光影、背景、输出形态），不拆成两份产物，不附独立设计报告。

## 压平输出

- 按类型文件"起草结构"的编号顺序合并为一段自然语言提示词。
- 删除标题、字段名、列表符号、占位符、代码栅栏和 JSON 语法。
- 短句、逗号连接，删除重复信息；固定块按下方两层契约并入。
- 内部资产编号（Cxx、Sxx、Pxx、Gxx 等）只用于项目管理、文件名、元数据、参考图绑定和审查记录，不得出现在交给图像模型的可复制提示词正文中；正文用自然名称指代。
- 只输出最终段落，不附结构、说明或分析。

## 两层固定块契约

- **几何契约（逐字）**：画面布局、比例标记、画面要求段的画幅、面板划分、视图、姿态、数量与完整入画要求逐字并入，不得改写。唯一例外是数量槽位：`[表情数]`、`[动作数]`、`[人数]` 等在起草时依据对应内容小节取**确数**（在类型文件给定范围内），最终提示词中不得出现区间。
- **呈现契约（功能固定，载体随媒介）**：光影、背景段中"均匀柔和""清晰可读""无戏剧性光效""纯净中性""无场景、道具、文字或装饰"等功能词逐字保留。"摄影棚"为载体词，仅适用摄影、实拍类媒介；其他媒介替换为等价中性载体（水墨→留白纸面，插画→平涂中性底，像素→纯色底），不得借替换引入新的风格信息。

## 锚点转译

只把 `regional_anchor` 与 `style_aesthetic` 中与当前资产相关的可见效果写进提示词，不粘贴抽象字段，不并置两个冲突块。地域真实优先于风格处理；风格不得抹除身份、解剖、服装构造、机制、尺度或世界事实。不描写所选视图之外的细节。

## 负面提示词

按当前资产、媒介、参考材料和真实生成风险现写；可从类型文件的评审清单中选取当次风险；不使用固定负面词清单，不堆叠无关限制。

## 跨资产依赖

- `Cxx-Lxx` 附对应 `Cxx` 基础参考图；缺图时给草案并标注身份锁未决。一个角色一个场景一份，不跨场景合并、不按单个动作拆分。
- `CSxx` 使用相关 `Cxx` 图与已确认的相对比例：同一图像尺度、同一地线，只比较不重新设计。
- `Gxx` 一份对应一种连贯群体功能；职业、阶层、年龄功能或社会角色需要实质不同的妆造系统时拆分，不混排。
- `Pxx` 一份对应一个道具的基准设计与基准状态；剧情关键的改变态（损坏、浸湿、耗尽、开启、激活）另立道具状态资产。
- `Sxx location_reference` 一份对应一个稳定地点、一个宽幅建立视角；除必要的匿名比例人物外保持空场景。
- 九宫格（multi-angle）仅在对应 `Sxx` 图确认后进行，以确认图为参考。

## 交付与平台设置

交付 16:9 资产页 / 参考图类生产提示词时，在元数据中附一行：

`平台设置：画幅比例 16:9（在生成平台侧设置；提示词文字不能改变画布比例）`

## 布局板参考图（可选）

生成平台支持参考图时，可将 assets/ 下对应类型的空白布局板（转为 PNG 后）以低权重构图参考随提示词提交，把布局约束从文本层提升到像素层；内容提示词不变。平台不支持参考图时跳过本节。
